Technical overview

  • IoT device in use :Cypress Quadro
  • IoT device is sending data(JSON) directly to a IoT Hub endpoint (via cloud gateway), hosted on Microsoft Azure. To connect, device is using MQTT library
  • Azure IoT Hub provides support for the AMQP 1.0 with optional WebSocket support, MQTT 3.1.1 and native HTTP 1.1 over TLS protocols
  • Data from IoT Hub are sent to Web App hosted on Azure via IoT Hub Endpoint
  • On the Web App we deploy/upload custom NodeJS Application from Git (HTML/CSS/JS files)
